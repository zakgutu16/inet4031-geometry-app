# inet4031-geometry-app

A simple Python Flask web application developed for INET 4031 at the University of Minnesota.

This app demonstrates basic web development using Flask, allowing users to calculate the **volume of cylinders and spheres** using both command-line prompts and dynamic web forms.

---

## 📁 Features

- Web-based UI for geometric calculations
- Flask framework with HTML templating
- Separate logic and unit tests for `cylinder.py` and `sphere.py`
- Flask routing for home, cylinder, and sphere pages

---
